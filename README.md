## English

### Description
This project is a C++ cheat program for CS2, featuring enemy pointers and radar functionality.

### Features
- pointer
- Radar
- 
### Requirements
- Windows OS
- Visual Studio 2019 or later
- C++17

### Build Instructions
1. Clone the repository:
```bash
git clone https://github.com/Akaline-tech/Cs2-External-Hack.git
```
2. Open the project in Visual Studio.
3. Set the build configuration to `Release` or `Debug`.
4. Build the solution (Ctrl + Shift + B).
5. Run the executable as Administrator if needed.

### Usage
- Press `INSERT` to toggle the menu.
- Press `F5` to toggle the pointer.
- Press `F6` to toggle the radar.
